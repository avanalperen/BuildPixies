StudyGroup Connect

1. Core Objective

Provide a seamless platform for university students to discover, join, and manage course-specific study groups to enhance collaborative learning.

2. Key Features (MVP)

  • Course Integration: Ability to search and filter groups based on standardized university course codes (e.g., CS101).
  • Group Discovery: Browsable directory of open study groups with current member count and next meeting time.
  • Real-time Chat: Basic text channel for each group to facilitate discussion and sharing quick links.
  • Shared Calendar: Integrated scheduling tool to propose and confirm meetup times (virtual or physical locations).
